## Deskripsi Tugas
Simulasi blockchain untuk penyimpanan hash bukti digital jaringan menggunakan Python. Program melakukan akuisisi file PCAP, perhitungan SHA-256, pembuatan blockchain, dan validasi integritas.

## Tools yang Digunakan
- Python 3.14
- Scapy 2.7.0
- Hashlib (built-in)
- VS Code

## Cara Menjalankan Program
```bash
python sourcecode\blockchain_pcap.py
```
